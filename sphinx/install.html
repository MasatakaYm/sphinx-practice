Sphinxのインストール¶

詳しくは公式のインストールガイドを参照のこと.

MacOS (brewを使う方法)¶

$ brew install sphinx-doc

インストール後、パスを通す

echo 'export PATH="/usr/local/opt/sphinx-doc/bin:$PATH"' >> ~/.zshrc

Linux (Ubuntu20.04)¶

$ sudo apt install python3-sphinx

PyPIからパッケージをインストールする¶

$ pip3 install sphinx
